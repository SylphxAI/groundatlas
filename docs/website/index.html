The problem is not missing docs. It is misplaced authority.

No tool

Knowledge stays in people, chat, stale docs, and agent guesses.

Generated wiki as truth

Useful prose slowly becomes an unverified parallel source of truth.

GroundAtlas

Generated maps stay useful because canonical truth remains in source-owned files.

Built for users, developers, and agents.

Users

Open one generated map and learn what to read first.

Developers

Run deterministic CLI checks locally and in CI without model keys.

Agents

Consume JSON/Markdown maps, inspect canonical sources, then validate changes.

Vendor-neutral by design.

One project control file

Use project.manifest.json to describe identity, truth homes, surfaces, commands, and adoption state.

Adapters, not lock-in

Existing ecosystem manifests can be detected as adapters. No AI provider, agent tool, or company doctrine is required.

Open-source promise.

  • Deterministic scan and audit path.
  • No network or model key required for core functionality.
  • Generated output is deletable and non-authoritative.
  • Future AI features must cite source-owned truth.
  • Npm install is advertised only after registry readback succeeds.
Started by SylphxAI, designed as a vendor-neutral open-source primitive for every team that wants safer humans and agents in codebases.
